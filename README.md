# 📈 株＋ - 投資シミュレーター

投資の勘を確かめるための仮想ポートフォリオ＆長期シミュレーションアプリ。

## 機能

- 💼 **仮想ポートフォリオ** - 購入日と金額を記録して損益を確認
- 📊 **長期シミュレーション** - 積立・一括投資の将来試算
- 📝 **メモ** - 銘柄への気づきや判断メモ
- 🔴 **仮想通貨リアル価格** - CoinGecko APIで自動取得（BTC/ETH/SOL/XRP）
- ⚠️ **最低購入金額警告** - 単元株・投信・仮想通貨の最低額チェック

## 対応商品

| 種類 | 例 |
|------|-----|
| 投信・インデックス | eMAXIS Slim S&P500、SBI-V など |
| 国内株 | トヨタ、ソニー、NTT など |
| 米国株・ETF | Apple、NVIDIA、VOO など |
| 仮想通貨 | BTC、ETH、SOL、XRP |

## セットアップ

```bash
npm install
npm run dev
```

## デプロイ（Vercel）

1. GitHubにpush
2. [vercel.com](https://vercel.com) でリポジトリを連携
3. 自動デプロイ完了
